# GO CHESS

Веб-проект про шахматы с несколькими форматами игры: онлайн-партии, игра против бота и локальные режимы на одном устройстве.

## Идея проекта (тезисно)

- Сделать шахматы доступными «в один клик» прямо в браузере без сложной настройки.
- Объединить в одном интерфейсе разные сценарии: онлайн-игра с людьми, тренировка и локальная игра.
- Дать игроку быстрый вход в партию через лобби и удобный набор игровых инструментов (история ходов, реванш, review).

## Основные механики

- **Лобби и комнаты**: создание партий, подключение по ссылке, переход к активным и завершённым играм.
- **Онлайн-режим**: синхронизация ходов между игроками, статусы, запросы на ничью/отмену хода, реванш.
- **Локальные режимы**:
  - игра против бота с выбором цвета и сложности;
  - локальный pass-and-play режим «Вдвоём» на одном устройстве.
- **Игровой цикл**:
  - подтверждение хода;
  - история ходов и PGN;
  - финальная модалка по завершению партии;
  - повторная партия (реванш).
- **Review mode**: навигация по полуходам с возвратом к текущей позиции.
- **Кастомизация и UX**: темы доски/интерфейса, наборы фигур, звуки, адаптивное отображение.
