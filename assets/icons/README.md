# Lobby SVG icons

Папка для иконок лобби GO CHESS: `assets/icons/`.

## Куда загружать новые иконки

Загружайте SVG-файлы строго в эту папку:

- `assets/icons/start-game.svg`
- `assets/icons/games.svg`
- `assets/icons/players.svg`
- `assets/icons/ichi.svg`
- `assets/icons/traninig.svg`
- `assets/icons/duo.svg`
- `assets/icons/education.svg` (зарезервировано на будущее)

## Требования к SVG

- `viewBox="0 0 32 32"`
- монохромный `fill="#BFFB4F"`
- прозрачный фон
- без inline-стилей и лишних stroke
