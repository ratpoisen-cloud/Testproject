# Sound assets (`assets/sounds`)

Этот файл описывает, какие звуки использует приложение, как их правильно называть и как безопасно обновлять аудио.

## Где настраиваются звуки

Основная конфигурация находится в `js/sound.js` в объекте `SOUND_DEFINITIONS`.

Каждое событие (`event`) ссылается на один или несколько файлов из `assets/sounds/`.

## Текущие обязательные имена файлов

Используйте **точно такие** имена (с учетом регистра):

### 1) Выбор фигуры
- `select.mp3` — звук выбора фигуры/клетки (`piece_select`).

### 2) Обычный ход
- `move-1.mp3`
- `move-2.mp3`
- `move-3.mp3`
- `move-4.mp3`

Эти 4 файла — варианты звука хода (`move`), проигрываются по очереди, чтобы звук не был однообразным.

### 3) Взятие (обычное)
- `capture-default-1.mp3`
- `capture-default-2.mp3`

Используются для события `capture_default`.

### 4) Взятие (дальний удар)
- `capture-ranged-1.mp3`
- `capture-ranged-2.mp3`

Используются для события `capture_ranged`.

## Зарезервированные события (пока без файлов)

В `js/sound.js` уже есть события, для которых можно добавить звуки позже:

- `castle` — рокировка
- `check` — шах
- `promotion` — превращение пешки
- `checkmate` — мат
- `game_start` — начало партии
- `game_end` — завершение партии
- `enemy_move` — ход соперника
- `your_turn` — ваш ход

Чтобы включить звук для такого события:
1. Добавьте аудиофайл(ы) в `assets/sounds/`.
2. Пропишите путь(и) в `src` соответствующего события в `js/sound.js`.

## Инструкция по редактированию звуков

1. Подготовьте новый звук в формате `.mp3`.
2. Сохраните файл с **точным именем** из списка выше.
3. Замените старый файл в `assets/sounds/`.
4. Очистите кэш браузера (или сделайте hard reload), чтобы услышать обновленный звук.
5. Проверьте в игре:
   - выбор фигуры,
   - обычный ход,
   - взятие,
   - дальнее взятие.

## Рекомендации по качеству

- Длительность: обычно 100–500 мс.
- Без длинной тишины в начале/конце.
- Умеренная громкость (без клиппинга/перегруза).
- Единый стиль всех игровых звуков (чтобы набор звучал цельно).

## Важно

В папке могут лежать старые/тестовые аудиофайлы. На работу приложения влияют только те файлы, которые перечислены в `SOUND_DEFINITIONS` в `js/sound.js`.
